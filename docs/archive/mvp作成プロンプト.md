# 測量前チェックシステム MVP作成プロンプト

## 🎯 作成依頼

以下の要件に基づいて、測量前チェックシステムのMVP（最小実用製品）をReactで作成してください。

---

## 📋 プロジェクト概要

### 目的
従来のNumbersベースのチェックシートをチャット形式のWebアプリケーションに置き換え、現場作業員が片手で簡単にチェックできるシステムを構築する。

### ターゲットユーザー
- 測量作業員（現場作業）
- 利用環境：iPhone/iPad/PC
- 屋外現場での使用（手袋装着の可能性あり）

---

## 🔧 技術要件

### 必須技術スタック
- **フロントエンド：** React（関数コンポーネント + Hooks）
- **スタイリング：** Tailwind CSS（レスポンシブ対応）
- **データ保存：** LocalStorage（ブラウザローカル）
- **PDF生成：** jsPDF（結果出力用）

### デバイス対応
- **モバイル：** iPhone/iPad（iOS Safari）
- **デスクトップ：** Chrome, Safari, Edge
- **レスポンシブ：** 320px〜2560px

- **はい/いいえ**の2つの大きなボタン

- 現在の進捗をパーセンテージとプログレスバーで表示

## 📊 チェック項目データ

### ①図面について（8項目）

| ID | 質問 | 備考 |
|----|------|------|
| item_01_01 | 使用する図面は最新データか | 工種の変わり目や施工箇所が変わるごとに確認 |
| item_01_02 | 使用する図面の縮尺は合っているか | 構造物等で寸法を確認 |
| item_01_03 | 使用する図面の既知点設定は済んでいるか | ・基準点のＸ・Ｙ座標が一致しているか確認<br>※許容誤差は２ミリ程度とする<br>※平面図はあくまでも参考図であり、線形等を確認する際に使用する<br>※平面図から座標を拾って測量するのはＮＧ |
| item_01_04 | 実際に現場で使用する材料の寸法を材料承諾図や布設図から確認しているか | ・平面図の材料寸法もあくまでも参考なので、実際入れる物で考える<br>・材料の寸法違いが測量位置・高さに影響する<br>※材料承諾図…材料メーカーよりもらう材料寸法のみの図面<br>※特に護岸ブロックは測量法線が変わることがあるので注意 |
| item_01_05 | 墨出し等の位置を出す際の図面の縮尺は正しいか | 寸法が旗揚げされている構造物等で寸法を確認する |
| item_01_06 | 設計データの線形が正しいか | 平面図に中心線形座標をプロットして確認する |
| item_01_07 | 既設の構造物との距離や高さの関係を野帳に書き込んでいるか | ・現場で測設した際に位置や高さなどの関係性をチェックする<br>・既設構造物は図面と一致しないこともあるので、事前に観測し確認しておく |
| item_01_08 | 仮基準点を設ける必要があるか | ・測定箇所と基準点の位置から判断する<br>・仮基準点設置時は９００の杭を使用し根入れを確保する<br>・基準点は仮であっても看板、囲いを設置する |

### ②基準点について（5項目）

| ID | 質問 | 備考 |
|----|------|------|
| item_02_01 | 測量業者が作成した基準点を使用しているか | 前年度使用している基準点でも測量業者による確認を行う |
| item_02_02 | 測量成果簿・水準成果から得た座標値・高さであるか | ・座標値を平面図にプロットしてみる<br>・基準点看板・囲いを設置し、座標・高さを確認できるようにする<br>※必ずＳＩＭＡデータを使用する<br>※座標Ｘ・Ｙは測量成果の数字、高さＨは水準成果の数字を使う |
| item_02_03 | 野帳・光波・快測ナビに基準点データを記録・登録したか | ※手書きまたはデータでも記録に間違いがないかダブルチェックする<br>※必ず測量前に記録しておく<br>※座標データを手入力するのは禁止（打ち間違いの元） |
| item_02_04 | 使用する基準点の配置は適切か | ・基準点同士の角度が３０～１５０°以内に収まっているか<br>・器械と基準点との距離が１００ｍ以内であるか確認する<br>(ＴＳ出来形計測時)(３級ＴＳは１００ｍ、２級ＴＳは１５０ｍ以内)<br>(杭ナビは３級以上と同等の性能)<br>・雑草等障害物を取り除く |
| item_02_05 | チェックする基準点（使用する基準点以外）を確認したか | ・必ず誤差を確認すること<br>※ここでは許容誤差　土工１０ミリ・その他５ミリ以内とする<br>・視準する点が器械点～視準点以上の距離ではいけない |

### ③計測・測設する位置の準備について（8項目）

| ID | 質問 | 備考 |
|----|------|------|
| item_03_01 | 測量対象構造物の前後の測点で勾配に変化はあるか | 事前に変化点があるかを調べて野帳に記録する |
| item_03_02 | 図面は所持・携帯しているか | ・３次元データが必ず合っているとは限らない<br>・図面を所持しいつでも確認出来るようにする |
| item_03_03 | 何測点設置するか計算簿を作成しているか | 事前に重機オペレータや作業員さんと打合せを行う |
| item_03_04 | 設置する丁張に対して何ｍおきに設置するか決めているか | 丁張間隔を事前に検討 |
| item_03_05 | 逃げ杭を打つ段取りはできているのか | 管路や構造物など通りの逃げ杭を出しておく |
| item_03_06 | チェック内容を確認して問題が無いか | 全体の流れを把握 |
| item_03_07 | 測量機器は正常に動作するか | バッテリー残量、キャリブレーション確認 |
| item_03_08 | 安全対策は十分か | 作業範囲の安全確認、保護具の着用 |

### ④器械設置時_既知点設置（8項目）

| ID | 質問 | 備考 |
|----|------|------|
| item_04_01 | 水平がとれているか | 気泡管の確認 |
| item_04_02 | 器械の転倒危険はないか | 三脚に重りを乗せる等、動かない対策をとる（土のうなど） |
| item_04_03 | 器械点と後視の距離は取れているか | 精度確保のため器械設置時はできるだけ遠くの点で行う |
| item_04_04 | 器械の位置は重機作業の邪魔にならないか | ・測量器械が損傷する危険がある<br>・器械設置のやり直しはかなりの時間ロスになる |
| item_04_05 | 選択した既知点は間違いないか | 視準する点を変えた後の確認（ダブルチェック） |
| item_04_06 | プリズムの高さ変更はないか | プリズムの高さ変更後の端末操作の確認 |
| item_04_07 | 急な天候不良に対応できるか | 雨に濡れない対策を準備している（袋など） |
| item_04_08 | 器械の脚をしっかりと踏み込み、地盤に固定しているか | ・必ず３点をしっかり踏み込んで固定する<br>・踏んでも入らない場合は場所を変えてみる<br>※夏季(舗装の上)、冬季の測量は器械の水平がずれやすい |

### ④器械設置時_後方交会法（6項目）

| ID | 質問 | 備考 |
|----|------|------|
| item_04b_01 | 器械と使用する２つの点の距離が等しいか | 使う基準点のおよそ中心あたりに器械を据える |
| item_04b_02 | プリズム高は端末側設定高さと合っているか | できれば器械設置時のプリズム高さを統一（例えばＨ＝０.４５） |
| item_04b_03 | 器械点、視準点で使った基準点以外の第３点で座標・高さを確認したか | 根本的な間違いがないかの最終チェック |
| item_04b_04 | 選択した既知点に間違いはないか | 器械上で選択した基準点で間違いはないか |
| item_04b_05 | 器械の位置は重機作業の邪魔にならないか | ・測量器械が損傷する危険がある（杭ナビだと約２００万円かかる）<br>・器械設置のやり直しはかなりの時間ロスになる<br>・掘削法肩から１ｍ以上の離隔を取って器械設置を行う |
| item_04b_06 | 器械の水平は狂っていないか | 快測ナビ使用時はスマホ画面右上の水平マークを確認 |

### ⑤計測時注意事項（8項目）

| ID | 質問 | 備考 |
|----|------|------|
| item_05_01 | ミラーの高さは設定と合っているか | スケールにて高さ確認 |
| item_05_02 | 器械点と後視以上の距離ではないか | 精度確保のためできるだけ近い距離での計測 |
| item_05_03 | 測設した座標間の距離などは合っているか | テープやスケールで計測してみる |
| item_05_04 | Front/BackとL/Rは合っているか | 方向の確認 |
| item_05_05 | レベルの読み値は正しいか | ・スタッフは前後にしっかりと振られているか声掛けをする<br>・スタッフの下（測定箇所）にゴミなど付着していないか<br>・最後に別の基準点に当たってみて誤差が無いか確認する |
| item_05_06 | 通りは合っているか | ・事前に出してある逃げ杭から光波を使用し通りが合っているのか確認<br>・逃げ杭と同一ライン上に来ているのか確認 |
| item_05_07 | 前視点、後視点で使った基準点以外の第３点で座標・高さを確認したか | 根本的な間違いがないかの最終チェック |
| item_05_08 | 丁張の設置後の通りは正しいか | ・丁張を掛けた後は丁張をのぞいてみて通りを確認する<br>・丁張を掛けた後は１０歩下がって全体を見て確認する |

## 📝 注意事項

### 重要な制約
1. **オフライン動作前提**
   - すべてのデータをLocalStorageに保存
   - 外部APIへの依存なし

2. **現場での使いやすさ**
   - ボタンは大きく
   - タップ操作を前提
   - 誤操作を防ぐ確認ダイアログ

3. **データの永続性**
   - ブラウザを閉じても続きから再開可能
   - LocalStorageのクォータ制限に注意（5-10MB）

### やらなくていいこと（フェーズ2以降）
- ユーザー認証
- クラウド同期
- 作業種類別の分岐
- 写真添付機能
- 複数チェックの履歴管理
