# PDF出力ロジック技術仕様・コードレビュー

## 1. 概要
本アプリケーションのPDF出力は、ブラウザ上でHTMLを一度画像化し、それをPDFドキュメントに貼り付ける「HTML-to-Canvas-to-PDF」方式を採用しています。
主な使用ライブラリは以下の通りです：
- **html2canvas**: DOM要素をCanvas（画像）に変換
- **jsPDF**: PDFファイルの作成および画像の埋め込み

---

## 2. PDF生成の仕組み（詳細）

### 2.1 仮想レンダリング
`ResultScreen.jsx` でPDF出力が実行される際、ユーザーが見ている画面とは別に、非表示のテンプレート `PDFTemplate.jsx` が DOM 上に配置されます（`left: -9999px`）。これにより、画面サイズやスクロール状態に左右されず、常に固定のレイアウト（A4縦を想定した 880px 幅）でPDFを構築できます。

### 2.2 重量（Weight）ベースの独自改ページロジック
Webブラウザの標準的な印刷機能では、テキストの途中でページが切れてしまうことが多々あります。本アプリではこれを解決するため、**「Weight（重み）システム」による手動改ページアルゴリズム**を実装しています。

1.  **各要素にWeightを割り当て**:
    - ヘッダー情報: 8
    - カテゴリ見出し: 2.5
    - 質問項目（入力なし）: 1.5
    - 質問項目（入力あり）: 2.5
2.  **積算と判定**:
    - 1ページあたりの上限（`PAGE_LIMIT = 30`）を定義。
    - 項目をループで回しながら Weight を加算し、上限を超える直前で新しい `div.pdf-page` 要素を作成します。
3.  **効果**:
    - 質問文の途中でページが変わることを論理的に防いでいます。

### 2.3 画像化とPDF化のパイプライン（`pdfGenerator.js`）
1.  `element.querySelectorAll('.pdf-page')` で分割されたページ要素をすべて取得。
2.  各ページに対して `html2canvas` を実行（高解像度化のため `scale: 2` を指定）。
3.  `jsPDF` を A4 サイズ（210mm x 297mm）で初期化。
4.  Canvas を JPEG データ（品質 0.95）に変換し、PDFの (0, 0) 座標に A4 横幅いっぱいで貼り付け。

---

## 3. 「途切れてしまう問題」の解決策
これまでの開発で、以下の工夫を重ねて「途切れ」や「文字欠け」を解消しています：

1.  **`minHeight` の動的制御**:
    かつては `min-height: 1244px` で固定されていましたが、コンテンツが少ない場合に広大な余白が生まれる要因となっていました。現在は削除し、内容に応じた高さでキャプチャし、PDFページ上部に寄せることで自然な見た目にしています。
2.  **`box-sizing: border-box` の徹底**:
    パディングが要素の外側に広がり、キャプチャ範囲から漏れるのを防ぐため、すべてのページ要素に `border-box` を適用しています。
3.  **DOMの完全静止状態でのキャプチャ**:
    `html2canvas` 実行時にレイアウトが崩れないよう、専用の隠しコンテナ内ですべてのスタイルをインラインまたはスコープ付きで適用しています。

## 4. コードレビュー

### 良い点 (Pros)
- **堅牢な改ページ制御**: 手動で Weight を計算しているため、ライブラリ任せの自動改ページよりも制御が正確です。
- **高解像度**: `scale: 2` を利用しており、印刷しても文字がぼやけにくいです。
- **ファイル名生成**: 現場名と日付を基に安全なファイル名を生成するユーティリティが統合されています。

### 改善の余地 (Cons)
- **Weightの静的管理**: 現在の Weight は固定値ですが、ユーザーが長いメモを入力した場合、想定以上の高さになりページ下部がわずかに溢れるリスクがあります。
  - *対策案*: メモの文字数に応じて Weight を動的に加算するロジックを検討可能です。
- **パフォーマンス**: ページ数が多い（例: 5ページ以上）場合、`html2canvas` の処理に数秒かかることがあります。

---

## 5. メンテナンス時の注意点
PDFのレイアウトを調整する場合は、`PDFTemplate.jsx` 内の `PAGE_LIMIT` または 各要素の Weight を調整してください。
画像解像度やファイルサイズを調整する場合は、`pdfGenerator.js` 内の `scale` や JPEG の品質値を変更してください。
